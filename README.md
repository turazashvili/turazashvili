## 🚀 Project: SchemaCraftAI

<p align="center">
  <a href="https://github.com/turazashvili/SchemaCraftAI">
    <img src="https://raw.githubusercontent.com/turazashvili/SchemaCraftAI/main/public/favicon.ico" alt="SchemaCraftAI Logo" width="120" />

**SchemaCraftAI** is an AI-powered platform to quickly design database schemas using natural language and automatically generate boilerplate code.

- 🔗 **Live Demo:** [schemacraftai.vercel.app](https://schemacraftai.vercel.app)
- 🌟 **GitHub:** [turazashvili/SchemaCraftAI](https://github.com/turazashvili/SchemaCraftAI)

**Key Features:**
- 🤖 **AI Schema Generation**
- 🎨 **Visual ER Diagrams**
- 🚧 **Boilerplate Code Generation**
- 💾 **Instant Project Export**
- 🚀 **One-click Deployments**
