## About Me

I'm the founder and CEO of [Vexrail](https://vexrail.com), an AI ad infrastructure company. We're building intent-based advertising for AI apps: think AdSense, but for chatbots, voice agents, and AI-powered products. No PII, no identity tracking. Ads are served based on what the user is looking for, not who they are.

Before Vexrail, I spent years building AI and blockchain products. I still write code every day. Most of my current stack runs on Claude Code with MCP servers handling everything from social media scheduling to analytics sync.

- Building [Vexrail](https://vexrail.com): intent-based ad infrastructure for AI apps
- [Antler](https://www.antler.co/) portfolio founder (Singapore)
- Writing the [Vexrail AI Ads Digest](https://www.linkedin.com/in/turazashvili) on LinkedIn
- Running a 4-platform social media pipeline automated with AI and MCP

## Projects

### Vexrail — Ad Infrastructure for AI Apps
The ad monetization layer for AI products. Intent-based targeting, no PII dependency, built for chatbots, voice agents, and AI-powered apps.

- **Website:** [vexrail.com](https://vexrail.com)

### IssueScout — Find Your First Open Source Contribution
An open source issue discovery platform that surfaces GitHub "good first issue" issues enriched with community health scores (0-100), AI difficulty estimation, and personalized recommendations. Built with Next.js 16, TypeScript, MongoDB, and OpenAI.

- **Live:** [issuescout.dev](https://issuescout.dev)
- **GitHub:** [turazashvili/issuescout.dev](https://github.com/turazashvili/issuescout.dev)

### SchemaCraftAI — AI-Powered Database Schema Design
